This repository contains information and analysis for the 2014 Divvy Data Challenge. Analysis notebook provided through Knitr in R. It can be viewed in any browswer and requires the folder "figure." For the script to run, R and the package knitr are required. Additional data is required, but has not been loaded for space considerations. GIS data from City of Chicago (http://www.cityofchicago.org/city/en/depts/doit/provdrs/gis.html)
